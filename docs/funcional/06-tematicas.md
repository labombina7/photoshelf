# Temáticas

Las **Temáticas** son colecciones personalizadas que el usuario puede crear para agrupar fotos por cualquier criterio subjetivo (proyectos, estilos, clientes, etc.).

## Características

- Cada temática tiene un **nombre** y un **color** de identificación
- Se muestran en el sidebar con su color y el número de fotos asociadas
- Filtrando por temática en la biblioteca se ven solo las fotos de esa colección

## Gestión

Desde el sidebar se pueden:
- **Crear** una nueva temática (nombre + color con selector)
- **Editar** el nombre o color de una temática existente (icono al pasar el ratón)
- **Eliminar** una temática (con confirmación)

## Asignar fotos

Desde el detalle de cada foto se pueden añadir o quitar temáticas. Una foto puede pertenecer a múltiples temáticas simultáneamente.

## Diferencia con Tags

| | Tags | Temáticas |
|---|---|---|
| **Propósito** | Describir el contenido | Agrupar colecciones |
| **Origen** | Manual o IA | Solo manual |
| **Color** | No | Sí |
| **Visibilidad** | Vista de Tags | Sidebar |
